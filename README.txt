Annotation processor for OSGi Declarative Service Components

This project contains an APT processor that generated Service Component Descriptions for DS.

For more information the see wiki:
https://github.com/javakontor/OSGi-Service-Component-Annotation-Processor/wiki

*Note:*
We have not released an "official" version yet. The DS Annotation processor is still in early beta status.
